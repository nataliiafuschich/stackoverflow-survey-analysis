# Stack Overflow Developer Survey Analysis

## Project Overview
This project presents an exploratory data analysis (EDA) of the Stack Overflow Developer Survey dataset. The goal was to identify trends in developer salaries, experience levels, programming language usage, and remote work distribution.

## Tools Used
- Python
- Pandas
- Jupyter Notebook

## Key Analysis Areas
- Salary distribution by country and industry
- Experience vs compensation analysis
- Programming language usage trends
- Remote work patterns

## Key Insights
- Identified salary differences across regions
- Determined correlation between experience and income level
- Highlighted Python adoption trends
- Extracted actionable insights from large-scale survey data
